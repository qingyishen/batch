    

               文件备份器V2.3(for WinXP only)      
         
                    Code by JM  2006-10-19 QQ:13132002


    网上的备份程序功能比较强大，但是却有个致命的缺陷：需要注册。如果你不理会，它会
三天两头罢工，给你脸色看；或者时不时弹出个窗口，问候你一下，让你烦不甚烦……使用备份
软件的代价是昂贵的，而不对重要文件作备份的话，意外操作之后数据的损失更是惨不忍睹的。
如果你对备份的要求不高，那么，你可以试一下这个软件。这个软件是免费的，并且，如果你
具备一定的DOS批处理编程能力的话，你还可以对源程序进行改动，使它能满足你的个性化需求。

    本程序具备如下功能：

        1、自由指定要备份的目录(或文件)、位置、备份时间和任务名称
        2、能检测输入是否正确
        3、能生成备份日志
        4、能多次添加备份任务
        5、能批量停止指定的备份任务
        6、能自动删除由本程序产生且不再使用的批处理程序和注册表信息
        7、能从备份文件中恢复文件到源文件中，并保持各文件的原始属性不变

    备份模式：
  
        当源文件有改动时，会在指定的时间点和每次开机的时候复制源程序到目的位置。

    注意：
　　　　1、本程序只能以管理员权限运行；
        2、本程序有bat和exe两种版本，bat版本可以用文本编辑软件打开，exe版本是在bat版本
    的基础上由QuickBFC编译而成，你可以用此软件把exe反编译为bat文件之后再编辑；
        3、QuickBFC可以在汉化新世纪(http://www.hanzify.org/)下载到;
        4、如果要对bat文件进行改动，请保留本人的版权字样，否则，备份程序有可能运行出错。
